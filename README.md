# ascii-dither-bg

Hosted asset for the ASCII dither background animation.

## Usage

Mount this script in any DOM element with the `data-ascii-dither-bg` attribute.

```html
<div data-ascii-dither-bg></div>
<script src="https://cdn.jsdelivr.net/gh/songch9511/ascii-dither-bg@main/ascii-dither-background.js"></script>
```

For Framer code components, use the absolute jsDelivr URL above as the `Script URL` property.

## Versioning

Pin to a specific version by replacing `@main` with a tag, e.g. `@v1.0.0`, to keep the jsDelivr cache stable across deploys.
